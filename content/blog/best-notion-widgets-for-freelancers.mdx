## How to Add a Blocs Widget to Notion

1. Open the relevant widget page on Blocs (e.g., [blocs.me/pomodoro-timer](https://blocs.me/pomodoro-timer))
2. Copy the embed URL for the widget (e.g., `https://blocs.me/pomodoro`)
3. In Notion, type `/embed` and select the Embed block
4. Paste the URL and click Embed
5. Resize the block to fit your page layout

That's it. No plugins, no Zapier, no API keys. See the full guide at [how to add widgets to Notion](https://blocs.me/blog/how-to-add-widgets-to-notion) if you run into any issues.
